A demo project using Flask + Redis deployed in Docker Swarm.

How it works

  • Web: Python Flask app, scaled with replicas
  • Redis: Shared cache storing page-view counter

Run it locally

git clone https://github.com/SobikaSobi/docker-swarm-counter.git
cd docker-swarm-counter
docker build -t counter:1.0 ./app
docker swarm init
docker stack deploy -c docker-compose.yml counter
